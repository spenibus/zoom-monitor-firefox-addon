zoom monitor

This is a firefox addon that allows the user to monitor the current zoom level.
It appears as a toolbar button.
A menu is also available to change the current level of zoom.

- "F" indicates full-page zoom.
- "T" indicates text-only zoom.
- The zoom selection list is based on the pref "toolkit.zoomManager.zoomValues".
